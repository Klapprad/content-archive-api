# content-archive-api

API-basiertes Tool zur automatisierten Archivierung von TikTok-Inhalten. Basierend auf yt-dlp (https://github.com/yt-dlp/yt-dlp). Mehrstufiges System gebaut zur systematischen Untersuchung von Social Media Content für die soziologische Forschung. 

Entwickelt für eine soziologische Abschlussarbeit zur systematischen Analyse von TikTok-Inhalten. Ermöglicht die Untersuchung von Social-Media-Content nach verschiedenen Aspekten wie bspw.:
- Thematische Schwerpunkte
- Narrative Strukturen
- Diskursive Muster


## Packages und Features
- **Backend**: Python Flask REST API
- **Filterfunktion** @username und Start-, Enddatum (YYYY-MM-DD)
- **Video-Download** via yt-dlp video extraktion
- **Transkription** (geplant für Textmining/Speech-to-Text)
- **Analyse** (geplant LLM Integration)
