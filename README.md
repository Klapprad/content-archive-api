# content-archive-api

API-basiertes Tool zur automatisierten Archivierung und Analyse von TikTok-Inhalten für wissenschaftliche Forschung, basierend auf yt-dlp (https://github.com/yt-dlp/yt-dlp/blob/master/README.md#usage-and-options). Mehrstufiges System zur systematischen Untersuchung von Social-Media-Content: 

Entwickelt für eine soziologische Abschlussarbeit zur systematischen Analyse von TikTok-Inhalten. Ermöglicht die Untersuchung von Social-Media-Content nach verschiedenen Aspekten wie bspw.:
- Thematische Schwerpunkte
- Narrative Strukturen
- Diskursive Muster


## Packages und Features
- **Backend**: Python Flask REST API
- **Filter**: für @username und Start-, Enddatum (YYYY-MM-DD)
- **Video-Download**: yt-dlp video extraktion
- **Transkription**: (geplant für Textmining/Speech-to-Text)
- **Analyse**: (geplant: LLM Integration)
